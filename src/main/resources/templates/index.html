Welcome to LearnWords

This app can help you to learn foreign words or phrases.

You should register to create your own data base. When you have an account you can create catalogs for words, update and delete them.

For adding words, you need create catalog open it and add new card with form.

To learn words, you need to open the catalog, make sure that there are cards in it, and click the "Learn" link, after which a page with cards opens.
